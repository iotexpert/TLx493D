\subsection{Detailed Description}
Low level driver for the T\+L\+V493\+D-\/\+A1\+B6. 

It simplifies the read and write operations when working with the internal registers of the sensor and also help prevent changes to the reserved data registers.

Write the register data from regmap to the sensor registers. 

If the {\bfseries regmap\+\_\+check} pointer points to a valid structure (is not N\+U\+LL), the reserved registers data from the {\bfseries regmap} will be overwritten with the reserved data from {\bfseries regmap\+\_\+check}. This ensures that the reserved data read from the sensor is properly written back to the sensor. This overwrite will need to happend only once, as the corrections will be stored inside {\bfseries regmap}.
