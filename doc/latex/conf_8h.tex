\subsection{Detailed Description}
Configure oversample parameters for T\+L\+V/\+T\+L\+E493D sensors. 

The values presented in this file represent exponent used to compute powers of two. If the value of some expression defined below if n, then 2$^\wedge$n frames will be used to create an oversampled frame.
